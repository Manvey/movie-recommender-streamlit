# movie-recommender-streamlit
This project is a fully functional AI-based Movie Recommendation System that suggests movies similar to the one selected by the user. It uses data preprocessing, natural language processing, feature engineering, similarity models, and live TMDB API integration to generate accurate and visually appealing recommendations.

**Live App:** [AI Movie Recommender](https://ai-movie-recommender-app.streamlit.app/)
